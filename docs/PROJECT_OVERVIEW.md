# Zeno

Zeno is a schema-driven structured text editor.

It edits configuration files through a universal Intermediate
Representation (IR), enforcing rules defined by declarative schemas.
Zeno is format-agnostic and extensible, designed to prevent invalid
configurations by design across YAML and other structured formats.

------------------------------------------------------------------------

## Philosophy

Core is dumb.\
Schema is smart.\
Adapters translate formats.\
UI renders from structure.

Zeno separates:

-   Structure (IR)
-   Rules (Schema)
-   Validation (Engine)
-   Format parsing (Adapters)

------------------------------------------------------------------------

## What Zeno Is

-   A universal structured document editor
-   Schema-authoring first platform
-   Format-agnostic core
-   Deterministic validation system

------------------------------------------------------------------------

## What Zeno Is Not

-   A runtime engine
-   A config executor
-   A YAML-only editor
-   A domain-specific tool
